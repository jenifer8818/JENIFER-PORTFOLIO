    Charge Pump

    The goal of this project was to design and implement a basic charge pump circuit demonstrating voltage doubling using capacitors and switches. Powered by a 5V DC supply, the circuit efficiently multiplies the input voltage, producing an output voltage ideally twice the input, across capacitor C2. Charge pumps like this are widely used in applications such as power supply voltage doubling, LCD biasing in portable devices, and flash memory programming.
